  • find()

    • Find method is used to get first value in an array that specifies the condition mentioned. For example if in an array we want to find the time at which an employee reached after 8am. So for this we can use find() and specify condition in its callback function. If no such result exist in the array the function returns 'undefined'

    See code in PollyFills.js

  • findIndex()

    FindIndex method is used to get first index in an array that specifies the condition mentioned. For example if in an array we want to find the index of first element, which is greater than a value we can use findIndex() and specify condition in its callback function. If no such result exist in the array the function returns -1

    See code in PollyFills.js
